# Ignite Lab | ReactJS

## Descrição

Aplicação desenvolvido durante o Ignite Lab - ReactJS, promovido pela [Rocketseat](https://www.rocketseat.com.br/). Plataforma de evento online com data para liberação e exibição de video aula.
